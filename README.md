# Swiftual

Swiftual is the framework package for the terminal UI controls, layout system,
terminal protocols, rendering primitives, workers, and optional TCSS support.

This package must not contain demo applications, demo TCSS files, or demo-specific glue.

## Package Layout

- `Sources/Swiftual`: reusable framework source.
- `Tests/SwiftualTests`: framework tests.
- `userDocs`: feature documentation and manual test checklists.
- `issues`: deferred bug reports and design investigations.

Sibling packages:

- `../SwiftualDemo`: pure Swift demo application.
- `../SwiftualTCSSDemo`: TCSS stylesheet demo application.
- `../RichSwift`: Rich-compatible rendering dependency.
